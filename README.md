# 基础服务

- 包含基于openid的登录逻辑 userService；
- 包含基础HttpClient功能服务httpClientService；
- 包含用户信息获取功能服务UserRoleService；
- 包含本地自定义状态变量服务globalStateService;
- 包含权限控制功能permissionGuard
- 基本 ng dev / ng build  等脚本封装
- environment基础配置


# 基础组件
- ant design组件
- shared模块（通用组件存放），预先放置了loading组件

# 使用方法
- git  clone项目到本地
- 根据项目名修改package.json 与angular.json中的项目名 name（保持一致 angualr.json中含有较多，建议全选修改）
